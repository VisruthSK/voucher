# Package index

## All functions

- [`add()`](https://visruthsk.github.io/voucher/reference/add.md) : Add
  a user to the vouched contributors list.
- [`check()`](https://visruthsk.github.io/voucher/reference/check.md) :
  Check a user's vouch status.
- [`denounce()`](https://visruthsk.github.io/voucher/reference/denounce.md)
  : Denounce a user by adding them to the VOUCHED file with a minus
  prefix.
- [`use_vouch()`](https://visruthsk.github.io/voucher/reference/use_vouch.md)
  : Initialize vouch in the current project
- [`vouch_gha()`](https://visruthsk.github.io/voucher/reference/vouch_gha.md)
  : Use vouch GitHub Actions
